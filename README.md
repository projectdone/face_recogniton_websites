# face_recogniton_websites
a web in masked face recognition system for undergraduate final test (本科毕设：可遮挡的人脸识别系统）

| 语言及框架  |   |
| :----  | :----  |
| Python  |  系统编写语言 |
| Falsk | Web框架，搭建浏览器接口 |

|  技术金字塔   |   |
|  :----  | :----  |
| 网络模型  | MobileNet |
| 损失函数 | Triplet Loss |
| 学习率优化 | SGD |
| 学习率调整 | warm up + Cosine annealing |
| 预训练模型 | FaceNet  |
